# Astrobot Telegram: тексты бота

Дата фиксации: 17 июня 2026.

## Назначение документа

Это черновик пользовательских текстов для MVP Telegram-бота. Тексты специально сделаны живыми, эмоциональными и с цветными символами, чтобы интерфейс ощущался теплее и увлекательнее.

После пробного запуска тексты можно редактировать по фактической реакции пользователей.

## Общий стиль

Бот говорит:

- мягко;
- уверенно;
- тепло;
- немного загадочно;
- без давления;
- без пугающих предсказаний;
- с практическим советом в конце.

Бот использует символы:

- ✨ для магии, личного инсайта, результата;
- 🌙 для лунных и мягких сообщений;
- 🔮 для прогнозов и разбора;
- 💫 для энергии периода;
- 💌 для любви и отношений;
- 💼 для работы;
- 💰 для денег;
- 🧭 для направления и выбора;
- 🪐 для астрологии;
- 🧩 для совместимости;
- 📝 для вопросов и данных;
- ⭐ для оплаты Stars;
- ✅ для успеха;
- ⚠️ для мягких предупреждений;
- 🕯️ для спокойного ожидания.

## Ошибки ввода

### Неверная дата

⚠️ Кажется, дата указана в другом формате.

Пожалуйста, напиши дату так: ДД.ММ.ГГГГ  
Например: 14.08.1996

### Неверное время

⚠️ Кажется, время указано в другом формате.

Пожалуйста, напиши время так: ЧЧ:ММ  
Например: 08:30

Если не знаешь точное время — нажми кнопку ниже.

Кнопки:

- Не знаю время рождения

### Слишком короткий вопрос

Хочу ответить точнее ✨  
Попробуй написать вопрос чуть подробнее.

Например: не просто "отношения?", а "Что мне важно понять в отношениях с этим человеком?"

### Слишком длинный вопрос

Вопрос получился очень объемным 📝  
Давай сфокусируем его в одну главную тему.

Напиши, пожалуйста, один вопрос, на который ты хочешь получить ответ в первую очередь.
