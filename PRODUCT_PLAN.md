# Astrobot Telegram: продуктовый план

## Принятое продуктовое решение

Мы создаем Telegram-бота для продажи автоматизированных астрологических и эзотерических услуг с генерацией ответов через искусственный интеллект.

Выбранные категории услуг:

1. Гороскоп-прогноз
   - На день
   - На неделю
   - На месяц
   - Любовь
   - Работа
   - Деньги

2. Совместимость с человеком
   - Романтическая совместимость
   - Эмоциональная динамика
   - Сильные стороны пары
   - Зоны конфликта
   - Советы для отношений

3. Натальная карта
   - Психологический портрет
   - Сильные и слабые стороны
   - Любовные сценарии
   - Карьера и деньги
   - Жизненное направление

4. Нумерологический прогноз
   - Анализ по дате рождения
   - Личное число
   - Энергия года, месяца или дня

5. Таро + астрология
   - Быстрый символический ответ
   - Расклад на отношения
   - Расклад на ситуацию
   - Совет по выбору

6. Лунный календарь
   - Благоприятные дни
   - Рекомендации для любви, красоты, денег и работы
   - Ежедневный лунный совет

7. Ответ на личный вопрос
   - Пользователь задает один вопрос
   - Бот дает структурированный AI-ответ в астрологическом стиле

## Направление MVP

Первая версия должна продавать и автоматически выдавать несколько понятных услуг внутри Telegram.

Рекомендуемые услуги для MVP:

1. Бесплатный мини-прогноз или мини-разбор
2. Платный гороскоп-прогноз
3. Платная совместимость
4. Платная натальная карта
5. Платный ответ на личный вопрос

Остальные направления можно добавить после того, как заработает основной путь заказа:

- Нумерология
- Таро + астрология
- Лунный календарь

## Основной путь пользователя в боте

1. Пользователь открывает бота.
2. Бот показывает короткое приветствие.
3. Пользователь выбирает услугу.
4. Бот собирает нужные данные.
5. Бот показывает цену и подтверждает заказ.
6. Пользователь оплачивает или отправляет подтверждение оплаты.
7. Бот генерирует AI-разбор.
8. Бот отправляет результат в Telegram.
9. Бот предлагает следующую релевантную услугу.

## Какие данные нужны от пользователя

Базовый профиль:

- Имя
- Дата рождения
- Время рождения, если известно
- Место рождения, если известно
- Пол, опционально

Дополнительные данные по услугам:

- Совместимость: имя второго человека, дата рождения, время и место рождения при наличии
- Личный вопрос: вопрос пользователя и жизненная сфера
- Таро + астрология: ситуация или вопрос, тип расклада
- Лунный календарь: тема и период

## Следующие шаги

1. Утвердить названия услуг и описания для клиента.
2. Определить цены и бесплатные лимиты.
3. Описать формы сбора данных для каждой услуги.
4. Написать AI-промпты для каждой услуги.
5. Спроектировать меню и сценарий Telegram-бота.
6. Реализовать MVP.
7. Протестировать полный путь: выбор услуги, оплата, генерация и выдача результата.
