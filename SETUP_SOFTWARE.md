# Astrobot Telegram: что установить для разработки

Дата фиксации: 17 июня 2026.

## Короткий вывод

Для старта нужно установить только базовый набор:

1. Python
2. Git
3. Visual Studio Code
4. Telegram Desktop или мобильный Telegram
5. Аккаунт Telegram и бот через BotFather
6. Доступ к AI API

Остальное можно поставить позже.

## Обязательно установить

### 1. Python

Нужен для разработки самого бота.

Рекомендация:

- Python 3.13.x, 64-bit
- Python 3.12.x тоже подойдет
- Самую новую 3.14.x можно не ставить для MVP, чтобы не ловить несовместимости библиотек

Скачать:

- https://www.python.org/downloads/windows/

При установке обязательно включить:

- `Add python.exe to PATH`
- `pip`

После установки проверить в PowerShell:

```powershell
python --version
pip --version
```

Если команда `python` не работает, проверить:

```powershell
py --version
```

### 2. Git

Нужен для сохранения версий проекта, откатов, истории изменений и будущей публикации на GitHub.

Скачать:

- https://git-scm.com/downloads/win

При установке можно оставлять настройки по умолчанию.

После установки проверить:

```powershell
git --version
```

### 3. Visual Studio Code

Нужен как редактор проекта.

Скачать:

- https://code.visualstudio.com/download

После установки желательно поставить расширения:

- Python
- Pylance
- GitLens, опционально
- Ruff, позже для проверки кода
- SQLite Viewer, опционально для просмотра базы

### 4. Telegram

Нужен для создания бота, тестирования сценариев и получения платежей Stars.

Скачать Telegram Desktop:

- https://desktop.telegram.org/

Также подойдет мобильный Telegram. Для регистрации аккаунта обычно удобнее мобильное приложение.

### 5. BotFather в Telegram

BotFather нужен, чтобы создать Telegram-бота и получить `BOT_TOKEN`.

Что сделать:

1. Открыть Telegram.
2. Найти `@BotFather`.
3. Написать `/newbot`.
4. Задать имя бота.
5. Задать username бота, который заканчивается на `bot`.
6. Сохранить токен.

Токен никому не отправлять в открытый чат и не публиковать.

### 6. AI API

Нужен ключ для генерации прогнозов и разборов.

На старте можно использовать:

- OpenAI API

Что понадобится:

- аккаунт;
- API key;
- возможность оплаты API-запросов;
- выбранная модель.

Ключ будет храниться в `.env` как `AI_API_KEY`.

## Желательно установить

### 1. DB Browser for SQLite

Удобная программа, чтобы смотреть базу заказов, пользователей и платежей без программирования.

Скачать:

- https://sqlitebrowser.org/dl/

Это не обязательно, потому что SQLite встроен в Python, но для просмотра данных очень удобно.

### 2. GitHub Desktop

Если не хочется работать с Git через команды, можно поставить визуальный клиент.

Скачать:

- https://desktop.github.com/

Это опционально. Для разработки хватит обычного Git и VS Code.

### 3. Windows Terminal

Если у тебя Windows 11, он обычно уже установлен.

Скачать, если нет:

- https://apps.microsoft.com/detail/9n0dx20hk701

Можно пользоваться обычным PowerShell, это нормально.

## Пока не устанавливать

### Docker

Понадобится позже, когда будем готовить стабильный деплой.

Для MVP на локальном компьютере не нужен.

### PostgreSQL

Понадобится позже для продакшена.

Для MVP используем SQLite.

### Redis

Понадобится позже для очередей, кэша и фоновых задач.

Для первой версии не нужен.

### Node.js

Для Telegram-бота на Python не нужен.

Может понадобиться позже, если будем делать веб-админку или лендинг.

## Что должно быть после установки

В PowerShell должны работать команды:

```powershell
python --version
pip --version
git --version
```

Или для Python:

```powershell
py --version
```

## Минимальный набор аккаунтов

Нужно иметь:

1. Telegram-аккаунт.
2. Бот, созданный через BotFather.
3. AI API аккаунт и ключ.

Позже:

4. GitHub-аккаунт.
5. Telegram Stars/Fragment для вывода Stars.
6. Аккаунт платежного провайдера, если добавим внешние оплаты.

## Важные файлы, которые появятся позже

### `.env`

В нем будут секреты:

```text
BOT_TOKEN=
ADMIN_TELEGRAM_ID=
AI_PROVIDER=
AI_API_KEY=
AI_MODEL=
DATABASE_URL=
```

Этот файл нельзя публиковать.

### `.env.example`

Пример без секретов. Его можно хранить в проекте.

## Рекомендуемый порядок установки

1. Установить Python.
2. Установить Git.
3. Установить VS Code.
4. Установить Telegram.
5. Создать бота через BotFather.
6. Получить AI API key.
7. Проверить команды в PowerShell.
8. Вернуться к разработке каркаса проекта.

## Если хочется совсем просто

Минимум для первого запуска:

1. Python
2. VS Code
3. Telegram
4. BotFather token
5. AI API key

Git можно поставить сразу, но даже если отложить на час, разработку все равно можно начать. Однако лучше установить Git сразу, чтобы проект с первого дня был под контролем версий.
